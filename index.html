MidGARD

Model‑Integrated Graph Analysis for Regression Derivation

MidGARD is a model‑driven test‑generation system designed in collaboration with SanDisk to improve regression testing for storage firmware. By transforming formal TLA+ specifications into structured, reproducible test sequences, MidGARD reduces manual engineering effort, increases coverage consistency, and strengthens firmware validation workflows across the development pipeline.
